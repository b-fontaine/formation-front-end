Partie 1 : Les sélecteurs simples

Les choses à faire:

  • Cibler les balises "p" et mettre le texte en couleur bleu

  • Cibler le pargraphe avec l'attribut id ayant pour valeur "im-alone" et mettre le texte en souligné
  • Sélectionner tous les noeuds du document et mettre le texte en majuscule

Partie 2 : Les sélecteurs d'attribut

Les choses à faire:

  • Mettre les bordures en pointillé les carrés ayant comme attribut "data-color"
  • Mettre en bleu les carrés ayant pour attribut "data-color" et qui contient dans leur valeur le mot "bleu"
  • Mettre en rouge le carré ayant exactement comme valeur "rouge"

Partie 3 : Les combinateurs

Les choses à faire:

  • Un retrait gauche de 20px pour tous les paragraphes descendants de la classe "exo-article".
  • Un background de couleur jaune pour les paragraphes frères directs du titre "h2" (qui se trouvent immédiatement après h2).
  • Une bordure noir et solide pour les paragraphes adjacents du titre "h2".
  • Une couleur verte pour les paragraphes fils des éléments "li".
